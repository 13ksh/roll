# 🎱 구슬 룰렛

> 마지막까지 살아남은 공이 당첨되는 물리 기반 추첨기

라이브 데모 → [test7.site](https://test7.site/)

---

## 이게 왜 특별함?

기존 랜덤 추첨기는 그냥 버튼 누르면 결과 나옴.  
이건 다름. **공이 직접 코스를 통과해야 당첨됨.**

- 위에 있다고 유리한 거 아님 → **역전 가능**
- 아래에 혼자 있어도 출구가 막혀있으면 못 나감 → **끝까지 긴장감**
- 회전 칼날, 핀, 좁은 출구 통과해야 살아남음

lazygyu 구슬 룰렛보다 **추첨 시간 빠름 + 긴장감 10배**

---

## 기능

- **물리 엔진** (Matter.js) — 실제 중력, 충돌, 반발력
- **회전 칼날** — 공을 튕겨내는 회전하는 벽
- **충격파** — 공이 2초 이상 멈추면 자동으로 폭발해서 튀김
- **자동 줌인** — 출구 구간에서 카메라 2배 확대
- **카메라 추적** — 탈락 전엔 맨 아래 공, 탈락 후엔 맨 위 생존 공 추적
- **실시간 등수** — 화면 우측에 현재 순위 표시
- **이름 입력** — 텍스트박스에 입력하면 즉시 공 생성
  - `1~34` → 1번부터 34번까지 자동 생성
  - `홍길동*3` → 홍길동 3개 (3배 확률)
- **벽 통과 방지** — 서브스텝 10회로 터널링 최소화
- **CONFIG 한 곳에서 다 커스텀** — 벽, 핀, 물리, 카메라 전부

---

## 사용

브라우저에서 `index.html` 열고 이름 입력 후 START.

```
홍길동 김철수 이영희   ← 띄어쓰기로 구분
1~10                   ← 1~10 자동 생성
팀A*5                  ← 팀A 5개
```
